# imgenc
The encryptor by any image file.

### Requirements
* Ubuntu 20.10
* Node.js

### Usage
**install**
```bash
npm install -g TakutoYoshikai/imgenc
```

**encrypt**
```bash
imgenc encrypt <SECRET FILE> <IMAGE FILE AS A KEY> <OUTPUT FILE>
```

**decrypt**
```bash
imgenc decrypt <ENCRYPTED FILE> <IMAGE FILE AS A KEY> <OUTPUT FILE>
```
